POSIX Interface - Abstraction Headers

These headers are included via
#include <posix/name.h>

Name
(includes) 		Description
provided Interface (Functions/Variables and Constants)
file.h File access and manipulation routines and necessary constants
open, close, write, read, fcntl, ioctl, errno
O_RDONLY, O_CREAT, ...; EFAULT, EINVAL, ...
socket.h
(file.h) 		generic Socket Interface
socket, connect, bind, listen, accept
AF_LOCAL, AF_INET, ...
inet.h
(file.h)
(socket.h) 		Internet (IPv4) access
gethostbyname, htons, htonl, ...
memory.h ANSI-C Memory allocation and manipulation routines
malloc, realloc, free, memcpy
string.h ANSI-C Strings
strlen, strcpy, strcmp, ...
signal.h POSIX signals
signal, sigaction, kill
SIGxxx
process.h
(signal.h) 		POSIX processes (threads will follow later)
fork, exec* (later: pthread_*)
